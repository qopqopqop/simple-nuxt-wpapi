## Setup

- Add `simple-nuxt-wpapi` dependency using yarn or npm to your project
- Add `simple-nuxt-wpapi` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    "simple-nuxt-wpapi",

    // With options
    [
      "simple-nuxt-wpapi",
      {
        /* module options */
      }
    ]
  ];
}
```
